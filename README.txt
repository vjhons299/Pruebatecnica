Instrucciones para Ejecutar Automatización con Selenium y Python
Este repositorio contiene un script de Python que automatiza algunos procesos de la página web https://test-qa.inlaze.com  tanto para el login como para el registro de usuarios utilizando Selenium.

Requisitos Previos:
- Python instalado en tu computadora. Puedes descargar Python desde python.org
- Instalar Selenium: pip install selenium
- Descargar el WebDriver para el navegador que desees utilizar (por ejemplo, ChromeDriver para Google Chrome). Asegúrate de que el WebDriver esté en el PATH del sistema.

Pasos para Ejecutar la Automatización:
- Clona o descarga este repositorio en tu computadora.
- Abre una terminal y navega hasta el directorio donde se encuentran los script de Python.
- Ejecuta el script de Python de la siguiente manera:
python nombre_del_script.py ejemplo: python ValidarRegistroCorrecto.py
- El script abrirá un navegador y realizará los pasos ingresados según el script seleccionado
- Asegúrate de tener una conexión a Internet estable para que el WebDriver pueda interactuar con la página web.
- Si encuentras algún problema durante la ejecución, revisa los mensajes de error en la consola para identificar posibles fallos.
